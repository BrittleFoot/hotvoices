# HOTVOICE

Плагин для Chrome добавляющий поддержку горячих клавиш сайту vk.com для отправки голосовых сообщений.

## КУДАЖАТЬЯНЕПАНИМАЮ

**Alt+V** пока что единственный доступный хот-кей он отвечает за начало записи и отправку её же.
